PUT /subscriptions/(account)/(name)¶

Updates an existing subscription.

Parameters

  • account – The account name.

  • name – The subscription name.

Status Codes

  • 201 Created – Created.

  • 400 Bad Request – Cannot decode json parameter list.

  • 401 Unauthorized – Invalid Auth Token.

  • 404 Not Found – Subscription Not Found.

  • 500 Internal Server Error – Internal Error.

GET /subscriptions/(account)/(name)¶
GET /subscriptions/(account)¶
GET /subscriptions/¶

Retrieves a subscription.

Parameters

  • account – The account name.

  • name – The subscription name.

Response Headers

  • Content-Type – application/x-json-stream

Status Codes

  • 200 OK – OK.

  • 401 Unauthorized – Invalid Auth Token.

  • 404 Not Found – Subscription Not Found.

  • 406 Not Acceptable – Not Acceptable.

  • 500 Internal Server Error – Internal Error.

Returns

Line separated list of dictionaries with subscription information.

POST /subscriptions/(account)/(name)¶

Creates a new subscription.

  • name – The subscription name.

Status Codes

  • 201 Created – Created.

  • 400 Bad Request – Cannot decode json parameter list.

  • 401 Unauthorized – Invalid Auth Token.

  • 404 Not Found – Subscription Not Found.

  • 500 Internal Server Error – Internal Error.

Returns

ID of newly created subscription.
