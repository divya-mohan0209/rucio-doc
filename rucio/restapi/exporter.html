GET /export/¶

Export data from Rucio.

Example request:

GET /export HTTP/1.1
Host: rucio.cern.ch

Example response:

HTTP/1.1 200 OK
Vary: Accept
Content-Type: application/json

{"rses": [{"rse": "MOCK", "rse_type": "TAPE"}], "distances": {}}
Response Headers

  • Content-Type – application/json

Status Codes

  • 200 OK – DIDs found

  • 401 Unauthorized – Invalid Auth Token

  • 406 Not Acceptable – Not Acceptable

Returns

dictionary with rucio data
