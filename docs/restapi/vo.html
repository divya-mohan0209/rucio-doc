Details

GET (/vos/()

List all VOs.

  • Response Headers
* [Content-Type](https://tools.ietf.org/html/rfc7231#section-3.1.1.5) – application/x-json-stream
  • Status Codes
* [200 OK](http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html#sec10.2.1) – VOs found.


* [401 Unauthorized](http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html#sec10.4.2) – Invalid Auth Token.


* [409 Conflict](http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html#sec10.4.10) – Unsupported operation.


* [500 Internal Server Error](http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html#sec10.5.1) – Internal Error.

  • Returns

    A list containing all VOs.

POST (/vos/(vo/recover()

Recover root identity for a given VO

  • Parameters
* **root_vo** – VO to be recovered.
  • Request JSON Object
* **identity** (*string*) – Identity key to use.


* **authtype** (*string*) – Type of identity.


* **email** (*string*) – Admin email for VO.


* **email** – Password for identity.


* **default** (*bool*) – Whether to use identity as account default.
  • Status Codes
* [201 Created](http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html#sec10.2.2) – VO recovered successfully.


* [401 Unauthorized](http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html#sec10.4.2) – Invalid Auth Token.


* [404 Not Found](http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html#sec10.4.5) – Account not found.


* [409 Conflict](http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html#sec10.4.10) – Unsupported operation.


* [500 Internal Server Error](http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html#sec10.5.1) – Internal Error.

POST (/vos/(vo)

Add a VO with a given name.

  • Parameters
* **new_vo** – VO to be added.
  • Request JSON Object
* **description** (*string*) – Desciption of VO.


* **email** (*string*) – Admin email for VO.
  • Status Codes
* [201 Created](http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html#sec10.2.2) – VO created successfully.


* [401 Unauthorized](http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html#sec10.4.2) – Invalid Auth Token.


* [409 Conflict](http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html#sec10.4.10) – Unsupported operation.


* [500 Internal Server Error](http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html#sec10.5.1) – Internal Error.

PUT (/vos/(vo)

Update the details for a given VO

  • Parameters
* **updated_vo** – VO to be updated.
  • Request JSON Object
* **description** (*string*) – Desciption of VO.


* **email** (*string*) – Admin email for VO.
  • Status Codes
* [200 OK](http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html#sec10.2.1) – VO updated successfully.


* [401 Unauthorized](http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html#sec10.4.2) – Invalid Auth Token.


* [404 Not Found](http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html#sec10.4.5) – VO not found.


* [409 Conflict](http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html#sec10.4.10) – Unsupported operation.


* [500 Internal Server Error](http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html#sec10.5.1) – Internal Error.
