Details

POST (/lifetime_exceptions/()

Create a new Lifetime Model exception.

  • Request JSON Object
* **dids** (*string*) – The list of dids.


* **pattern** (*string*) – The pattern.


* **comments** (*string*) – The comment for the exception.


* **expires_at** (*string*) – The expiration date for the exception.
  • Response Headers
* [Content-Type](https://tools.ietf.org/html/rfc7231#section-3.1.1.5) – application/json
  • Status Codes
* [201 Created](http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html#sec10.2.2) – Created.


* [400 Bad Request](http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html#sec10.4.1) – Cannot decode json parameter list.


* [401 Unauthorized](http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html#sec10.4.2) – Invalid Auth Token.


* [409 Conflict](http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html#sec10.4.10) – Lifetime Exception already exists.


* [500 Internal Server Error](http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html#sec10.5.1) – Internal Error.

  • Returns

    The id for the newly created execption.

GET (/lifetime_exceptions/()

Retrieve all exceptions.

  • Response Headers
* [Content-Type](https://tools.ietf.org/html/rfc7231#section-3.1.1.5) – application/x-json-stream
  • Status Codes
* [200 OK](http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html#sec10.2.1) – OK.


* [401 Unauthorized](http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html#sec10.4.2) – Invalid Auth Token.


* [404 Not Found](http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html#sec10.4.5) – Lifetime Exception Not Found.


* [406 Not Acceptable](http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html#sec10.4.7) – Not Acceptable.


* [500 Internal Server Error](http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html#sec10.5.1) – Internal Error.

GET (/lifetime_exceptions/(exception_id)

Retrieve an exception.

  • Parameters
* **exception_id** – The exception identifier.
  • Response Headers
* [Content-Type](https://tools.ietf.org/html/rfc7231#section-3.1.1.5) – application/x-json-stream
  • Status Codes
* [200 OK](http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html#sec10.2.1) – OK.


* [401 Unauthorized](http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html#sec10.4.2) – Invalid Auth Token.


* [404 Not Found](http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html#sec10.4.5) – Lifetime Exception Not Found.


* [406 Not Acceptable](http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html#sec10.4.7) – Not Acceptable.


* [500 Internal Server Error](http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html#sec10.5.1) – Internal Error.

  • Returns

    List of exceptions.

PUT (/lifetime_exceptions/(exception_id)

Approve/Reject an execption.

  • Parameters
* **exception_id** – The exception identifier.
  • Request JSON Object
* **state** (*string*) – the new state (APPROVED/REJECTED)
  • Status Codes
* [201 Created](http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html#sec10.2.2) – Created.


* [400 Bad Request](http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html#sec10.4.1) – Cannot decode json parameter list.


* [401 Unauthorized](http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html#sec10.4.2) – Invalid Auth Token.


* [404 Not Found](http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html#sec10.4.5) – Lifetime Exception Not Found.

  • Status 500

    Internal Error.
