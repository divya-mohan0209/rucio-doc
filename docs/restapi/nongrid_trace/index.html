Details

POST (/nongrid_traces/()

Trace endpoint used by the XAOD framework to post data access information.

  • Request JSON Object
* **payload** (*dict*) – Dictionary contain the trace information.
  • Status Codes
* [201 Created](http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html#sec10.2.2) – Created.


* [400 Bad Request](http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html#sec10.4.1) – Cannot decode json data.


* [500 Internal Server Error](http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html#sec10.5.1) – Internal Error.
