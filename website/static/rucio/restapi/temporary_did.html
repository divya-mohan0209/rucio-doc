POST /tmp_dids/¶

Addition of temporary data identifiers in bulk.

Request JSON Object

  • dids (list) – A list of dids.

Status Codes

  • 201 Created – Created.

  • 400 Bad Request – Cannot decode json parameter list.

  • 401 Unauthorized – Invalid Auth Token.

  • 500 Internal Server Error – Internal Error.
