GET /requests/list¶

Lists requests for a given source and destination RSE or site.

Request Headers

  • Content-Type – application/x-json-stream

Status Codes

  • 200 OK – Request found.

  • 404 Not Found – Request not found.

  • 406 Not Acceptable – Not Acceptable.

GET /requests/(scope)/(name)/(rse)¶

Lists request for given DID to a destination RSE.

Parameters

  • scope – data identifier scope.

  • name – data identifier name.

  • rse – destination RSE.

Request Headers

  • Content-Type – application/json

Status Codes

  • 200 OK – Request found.

  • 404 Not Found – Request not found.

  • 406 Not Acceptable – Not Acceptable.
