Account Limit Rest API¶

Overview

Resource

Operation

Description

AccountLimit

POST /accountlimits/(account)/(rse)

Create/update account limits.

DELETE /accountlimits/(account)/(rse)

Delete account limits.

Details

POST /accountlimits/(account)/(rse)¶

Create or update an account limit.

Parameters

  • account – The account identifier.

  • rse – The rse name.

Status Codes

  • 201 Created – Successfully created or updated.

  • 401 Unauthorized – Invalid auth token.

  • 404 Not Found – RSE not found.

  • 404 Not Found – Account not found

DELETE /accountlimits/(account)/(rse)¶

Delete an account limit.

Parameters

  • account – The account identifier.

  • rse – The rse name.

Status Codes

  • 200 OK – Successfully deleted.

  • 401 Unauthorized – Invalid auth token.

  • 404 Not Found – RSE not found.

  • 404 Not Found – Account not found
